# Lintu Delivery Game

Браузерная мини-игра для сайта сервиса доставки дронами **Lintu**. Статический проект без сборки и зависимостей.

## Запуск

Откройте файл `index.html` в браузере или поднимите любой статический сервер в корне проекта.

## Встраивание на сайт

Скопируйте папку проекта на хостинг и вставьте игру через `<iframe>` или подключите разметку и файлы `style.css` / `script.js` в свой шаблон.

## Структура

- `index.html` — разметка и экраны
- `style.css` — стили
- `script.js` — логика игры
- `assets/` — изображения (дроны, фон карты)
